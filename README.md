# HaloChat

**HaloChat** is a modern, **AI-powered lightning-fast, real-time chat application** built for scalability, responsiveness, and seamless instant messaging. It combines real-time messaging, smart AI features, and a clean, intuitive user experience — ideal for both one-on-one chats and group conversations. The goal is to provide users with a powerful communication platform and developers with a modular, easy-to-extend codebase.

## 🌟 Features

- 💬 Real-time messaging with instant updates (WebSockets)
- 🤖 AI-assisted chat capabilities
- 👥 User accounts and customizable profiles
- 👀 Typing indicators & presence detection
- 📤 File/image upload support
- 🌓 Dark mode toggle
- 📱 Fully responsive design
- 🔔 Push notifications for new messages
- 📁 Group chats & private channels
- 🚀 Fast and responsive UI
- ⚙️ Secure, Clean, scalable code structure

## ⚙️ Local Setup and Installation

1. **Install dependencies:**

   ```bash
   pnpm install
   ```

2. **Environment Setup:** Check the **`.env.sample`** file and configure your **`.env`** with necessary API keys and database connections.
3. **Start the app:**

   ```bash
   pnpm run dev
   ```

   The application will be accessible at `http://localhost:5173`.
